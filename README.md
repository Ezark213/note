# 借金キャラ SNS × note マネタイズ戦略

借金をテーマにした自虐系SNSアカウント運用と、noteでの有料コンテンツ販売を組み合わせたマネタイズ戦略をまとめたリポジトリです。

## 概要

実際の借金経験をもとに、X（Twitter）で共感を集め、noteで有料エッセイ・ノウハウを販売するための包括的な戦略ドキュメントです。

## コンテンツ

### [借金キャラSNS×noteマネタイズ戦略.md](./借金キャラSNS×noteマネタイズ戦略.md)

- **対象ペルソナ** - ターゲット読者の詳細設計
- **アカウント設計** - 名前案、プロフィール、固定ポスト
- **SNS投稿戦略** - カテゴリ設計、バズ狙いテンプレ、通常運用テンプレ
- **note商品設計** - 記事タイトル案10本、構成、導入文テンプレ
- **投稿頻度・運用導線** - 初期〜安定期の運用計画
- **失敗パターン** - 伸びないアカウントの共通点と回避策

## コンセプト

> 「努力した結果、借金が増えました」

成功談ではなく、リアルな失敗談と延命策を発信。同じ境遇の人に「自分だけじゃない」という安心感と、実用的なサバイバル知識を提供します。

## 戦略のポイント

| 要素 | 内容 |
|------|------|
| キャラ設定 | 自虐・諦観ベースの借金キャラ |
| 発信軸 | 成功談ゼロ、延命策と失敗談のみ |
| 収益化 | note有料記事（数百円〜） |
| ターゲット | 20〜30代、借金300万円規模の社会人 |

## 注意事項

- 過度なポジティブや成功アピールは避ける
- 他者攻撃・同情乞いはNG
- 事実ベースで、過度な誇張はしない
- プラットフォーム規約を遵守する
